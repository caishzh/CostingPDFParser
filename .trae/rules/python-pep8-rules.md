# Python 代码规范（强制）
language: Python
alwaysApply: true  # 所有Python代码自动生效

## 一、空行规范（核心）
- 空行必须是**纯空白行（0个空格/0个Tab）**，禁止任何空格残留
- 空行**不参与缩进**，无需与上一行/上一级对齐
- 顶层函数/类之间：2个空行
- 类内方法之间：1个空行
- 函数内逻辑块之间：1个空行（按需）

## 二、缩进与格式
- 缩进：4个空格，禁用Tab
- 行宽：≤79字符
- 命名：变量/函数用snake_case；类用PascalCase；常量全大写
- 注释：# 后加1空格；行内注释与代码间隔≥2空格

## 三、格式化要求
- 所有代码生成/修改后，自动按PEP8格式化
- 自动删除空行内所有空格
- 自动统一缩进、空行、命名
